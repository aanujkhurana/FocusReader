# ORP Centering Test

This document contains words of different lengths to test the ORP (Optimal Recognition Point) centering:

- One letter: I A
- Two letters: is to be or at in on up we me
- Three letters: the and but for you can see how now
- Four letters: word this that with from they have been
- Five letters: hello world these words should center
- Six letters: proper center should happen always
- Seven letters: reading becomes smoother focused
- Eight letters: elephant keyboard computer software
- Longer words: extraordinary implementation functionality

The red highlight should now be properly centered on each word, especially for three-letter words like "the", "and", "but", "for", "you", "can", "see", "how", and "now".
